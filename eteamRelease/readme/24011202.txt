OPEN21 Workflowの修正版モジュールを送付します。


【アップデート手順】
以下の手順にて、OPEN21 Workflowサーバーに対してモジュールのアップデートを行ってください。

１．念の為、DBのバックアップを取得してください。
	・OPEN21 Workflowサーバーで、下記バッチを管理者権限で実行してください。
		C:\eteam\bat\bin\backup.bat
		
２．ミドルウェアバージョンアップ
　　・「ワークフローミドルウェアアップデート手順」に沿って作業してください。
　　　　　　Apache HTTP Server：2.4.58-VS17
　　　　　　Apache Tomcat：9.0.84
　　　　　　AdoptOpenJDK：11.0.21+9
　　　　　　VC_redist.x64：14.36.32532.0

３．一式デプロイ
	・C:\eteam フォルダを退避してください。
	・zip\eteam_install\eteam フォルダを、OPEN21WFサーバーのCドライブ直下に上書きコピーしてください。

４．モジュールアップデート
	・OPEN21WFサーバーで、下記バッチを管理者権限で実行してください。
		C:\eteam\patch\update.bat
